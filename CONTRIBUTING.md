# Contributing to Naruto Anime Portfolio

Thank you for considering contributing to the Naruto Anime Portfolio project! Your contributions are welcome and appreciated. Here are some guidelines to help you get started:

## Fork the Repository
1. Go to the [Naruto Anime Portfolio GitHub repository](https://github.com/[owner]/naruto-anime-portfolio).
2. Click on the **Fork** button at the top right of the page to create your own copy of the repository.

## Create a New Branch
1. After forking the repository, clone your forked version to your local machine:
   ```bash
   git clone https://github.com/[your-username]/naruto-anime-portfolio.git
   ```
2. Navigate into the project directory:
   ```bash
   cd naruto-anime-portfolio
   ```
3. Create a new branch for your feature or fix:
   ```bash
   git checkout -b your-feature-branch-name
   ```

## Make Your Changes
1. Make the necessary changes in your code.
2. Add and commit your changes:
   ```bash
   git add .
   git commit -m "Description of your changes"
   ```

## Submit a Pull Request
1. Push your changes to your forked repository:
   ```bash
   git push origin your-feature-branch-name
   ```
2. Go to the original repository where you want to submit your changes.
3. Click on the **Pull requests** tab.
4. Click on **New pull request**.
5. Select your forked repository and the branch you just pushed.
6. Add a title and description for your pull request, then click on **Create pull request**.

## Additional Notes
- Ensure that your code follows the existing code style.
- Test your changes thoroughly.
- Provide clear commit messages for each change you make.

Thank you for contributing! We value your effort and time spent on this project!
