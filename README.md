# Naruto Anime Portfolio

This project is a complete portfolio application for Naruto fans featuring a Frontend, Backend, AI Service, and more. Below are the built features:

## Features
- **Frontend**: Built using Next.js 14, React 18, and TypeScript.
  - Pages for Episodes, AI Generator, Profile, and Chat.
  - 4K streaming with Naruto Classic and Shippuden in separate tabs.
  - Google OAuth login system.
  - Custom avatar generation and profile customization.
  - Font size, color, and anime-style font customization.
  - Real-time chat with Naruto emojis.
  - Ad system for monetization and a credit system for monetized content generation.

- **Backend**: Node.js/Express server with:
  - Routes, Controllers, Models, Middleware, and Services.

- **AI Service**: Python FastAPI application for video and image generation.

- **Docker**: Docker Compose setup for local development.

## Installation
1. Clone the repository
2. Install dependencies for both frontend and backend
3. Start the services using Docker
